“匿名委托”示例

本示例演示了如何使用匿名委托来计算员工的薪水奖金。使用匿名委托简化了程序，因为无需再定义一个单独的方法。

每个雇员的数据都存储在一个对象中，该对象包含个人详细信息和一个引用计算奖金所需的算法的委托。通过以委托的方式定义算法，可以使用相同的方法来执行奖金计算，而与实际计算方式无关。另外需要注意的是，一个局部变量 multiplier 变成了已捕获的外部变量，因为它在委托计算中被引用了。
