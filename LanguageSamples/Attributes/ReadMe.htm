“特性”示例

本示例演示了如何创建自定义特性类，如何在代码中使用这些类，以及如何通过反射查询它们。有关特性的其他信息，请参见特性（C# 编程指南）.

在 Visual Studio 中生成并运行“特性”示例

  • 在“调试”菜单上，单击“开始执行(不调试)”。

从命令行生成并运行“特性”示例

  • 在命令提示符处，键入以下命令：

    csc AttributesTutorial.cs
