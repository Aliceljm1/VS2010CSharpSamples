注意：

本示例在 Microsoft Office Outlook 2007 及更高版本中运行。

本示例演示如何使用 Microsoft Office Outlook 窗体区域显示来自 RSS 源的文章的完整内容。 Outlook 通常只显示标题新闻。

在阅读窗格中查看某个 RSS 项时，一个相邻的窗体区域会追加到该项的底部。

在 Outlook 检查器中查看 RSS 项时，一个单独的窗体区域会作为单独页面显示在该项中。

运行此示例

  1. 按 F5。

    Outlook 打开。

  2. 打开一个 RSS 项。 有关更多信息，请参见“阅读 RSS 源”。

    注意：

    您必须已订阅 RSS 源。 有关更多信息，请参见“添加 RSS 源”。

  3. 在消息列表中选择一个消息。

    该消息会显示在阅读窗格中。 完整文章会显示在该消息下方，位于一个相邻窗体区域中。

  4. 双击消息列表中的一个消息。

    该消息会在一个检查器窗口中打开。

  5. 在功能区的“显示”组中，单击“View Article”（查看文章）。

    完整文章会显示为窗体中的新页面。

  6. 在该页面的顶部，单击“Search for Similar Topics”（搜索相似主题）。

    “Live Search”（动态搜索）网站会打开并显示与文章标题相关的搜索结果。

  7. 在页面顶部，单击“Hide Search Results”（隐藏搜索结果）。

    搜索结果在页面中将不再可见。

要求

本示例需要如下内容：

  • Visual Studio Tools for Office。

  • Microsoft Office Outlook 2007。

  • 订阅 RSS 源。 有关如何在 Outlook 中订阅 RSS 源的信息，请参见“添加 RSS 源”。

  • 在 Outlook 中必须打开阅读窗格。 有关更多信息，请参见“打开或关闭阅读窗格”。

演示

本示例演示以下概念：

  • 创建单独和相邻的窗体区域。

  • 在窗体区域中使用托管控件。

  • 访问 Microsoft.Office.Interop.Outlook.PostItem 类的实例。 此类表示用于显示窗体区域的 RSS 项。

  • 在 Microsoft.Office.Tools.Outlook.FormRegionControl.FormRegionShowing 事件处理程序中使用 Microsoft.Office.Interop.Outlook.PostItem 实例，以便在窗体区域出现在 Outlook 中之前对其进行处理。
