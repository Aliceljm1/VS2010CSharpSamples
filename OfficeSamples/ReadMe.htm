以下示例旨在展示 Visual Studio 2010 的 Office 开发功能。这些示例面向 Office 2007、Office 2010 或更高版本。

若要获得最佳体验，请加载与您的 .NET Framework 目标版本相对应的示例版本。 这些示例组织到以下子目录中：

\Office
    \FX40 - 为在 .NET Framework 4.0 Client Profile  [Visual Studio 2010 的默认设置] 上运行而进行了优化的示例
    \FX35 - 为在 .NET Framework 3.5 上运行而进行了优化的示例；通过进行下面介绍的少许更改，这些示例可以修改为在 4.0 上工作

有关针对 3.5 和 4.0 进行的 Office 开发之间的差异的更多信息，请参见以下主题：
将 Office 解决方案迁移到 .NET Framework 4
