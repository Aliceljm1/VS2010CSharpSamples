Microsoft WCF 数据 服务应用程序示例

此示例演示一个简单的 WCF 数据服务，它通过 ADO.NET Entity Framework 公开数据。 此外还演示了一个使用该 WCF 数据服务的 WPF 应用程序。

该解决方案由以下项目组成：

DataServicesWebApp

此项目包含将用作 WCF 数据服务的基础的 实体数据模型，以及该数据服务本身。

UserInterface

此项目是在 WPF 中实现的用户界面。应用程序的入口点 位于 ProductList.xaml 的代码隐藏文件中。
